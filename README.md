* 请在 `scripts` 文件夹下新建文件夹，文件夹名为脚本用途概述；
* 文件夹内部请用 `README.md` 简单介绍脚本使用方法，markdown 中图片资源请放在文件夹中；
* 新建脚本后，请修改根目录 `README.md` 中的 `脚本列表`。
